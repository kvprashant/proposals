Proposals
=========

Proposals system for Devthon hack tracks

This code runs on the [Meteor](http://meteor.com) framework. All you have to do before checkout is, install meteor by:

    $ curl https://install.meteor.com | /bin/sh

Then, after checkout:

    $ cd proposals
    $ meteor
